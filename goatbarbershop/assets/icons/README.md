# Ícones Sociais

## Status Atual
Os ícones sociais foram temporariamente substituídos por ícones do Material Design para evitar erros de carregamento.

## Ícones Necessários
Para uma melhor experiência visual, adicione os seguintes arquivos PNG (24x24px ou 48x48px):

- `google.png` - Ícone do Google
- `apple.png` - Ícone da Apple  
- `facebook.png` - Ícone do Facebook

## Onde Baixar
- **Google**: [Brand Guidelines](https://developers.google.com/identity/branding-guidelines)
- **Apple**: [Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/sign-in-with-apple)
- **Facebook**: [Brand Guidelines](https://developers.facebook.com/docs/facebook-login/userexperience/)

## Alternativas Gratuitas
- [Icons8](https://icons8.com)
- [Flaticon](https://www.flaticon.com)
- [Feather Icons](https://feathericons.com)

## Como Usar
1. Baixe os ícones nos formatos PNG
2. Renomeie para `google.png`, `apple.png`, `facebook.png`
3. Coloque nesta pasta (`assets/icons/`)
4. Execute `flutter clean` e `flutter pub get`
5. Reinicie a aplicação

## Fallback Atual
Enquanto os ícones não são adicionados, o app usa:
- Google: ícone de conta (account_circle)
- Apple: ícone nativo do Material Design
- Facebook: ícone nativo do Material Design
