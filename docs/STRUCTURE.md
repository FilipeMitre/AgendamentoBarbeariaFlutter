# Project Structure

This document outlines the structure of the Agendamento Barbearia Flutter project.

## Directory Structure
- **lib/** : Contains all the Dart files for the application.
- **docs/** : Documentation files.
- **db/** : Database schemas and migrations.
- **.github/** : GitHub-related files, including issue templates and pull request templates.

## File Descriptions
- **lib/app.dart**: Main application entry point.
- **lib/main.dart**: Used to run the app.
- **lib/utils/constants.dart**: Includes common constants used throughout the app.
- **lib/widgets/**: Contains reusable widgets for the application.

## SQL Database Schema
- **db/schema.sql**: Contains SQL scripts for creating and initializing the database schema.

Ensure to follow proper Git workflow for contribution and code reviews.
